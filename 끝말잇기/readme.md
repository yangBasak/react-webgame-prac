## 끝말잇기

* 여기서부터는 node와 npm을 이용하여 react를 설치해서 사용한다.

* webpack을 설치한다.

* 파일 생성
### index.html
단 하나의 HTML 파일
### client.jsx
만들어진 js 파일들을 html DOM에 붙이는 작업하는 곳. 
### wordChainGame.jsx
컴포넌트들 중 하나. 이런 애들이 여럿 만들어져서 client.jsx에 모여 실행된다
### webpack.config.js
HTML 파일 내의 script는 하나만 존재하는데, 위의 작성된 모든 js 파일들을 하나로 만들어주는 곳이다.
